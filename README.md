# Lighthouse

> Lighthouse is a PCB that controls lights in my house.

<img width="10%" src="./logo.svg">



## Features

- ESP8266 module (works with ESPHome)
- Two constant voltage outputs and two constant current outputs
- Powered from USB-C, barrel, or screw terminal
- Voltage outputs can be powered separately, or from global USB/barrel/screw terminal

It's doesn't have to be for lighting, but any small load you wish to switch remotely.
